## Implemented Features
### Custom Instructions
- **`MOD DST, SRC`** - Performs modulus operation: `DST = DST % SRC`  
- **`CMP0 SRC`** - Compares `SRC` with zero and sets the Zero Flag (ZF) accordingly  

### Algorithms
- **CHECKPRIME** — Determines if a number `X` is prime using `MOD` and `CMP0` instructions.  
- **GETDIVCOUNT** — Calculates how many times `Y` divides `X` until no longer divisible.
